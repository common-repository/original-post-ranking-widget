== Description ==

To view the post ranking in the form of a widget.
Display thumbnails and category.
And also customize the tag.
This plugin is a plug-in that Customize the Original Post ranking.

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Open the plugin configuration page,
   which is located under Options -> Original Post Ranking Widget
